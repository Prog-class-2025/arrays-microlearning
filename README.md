# 🎓 Практична робота: Масиви (C++/CLI, Windows Forms)

## 📌 Мета
Навчитися працювати з:
- одновимірними масивами
- двовимірними масивами (матрицями)
- генерацією випадкових чисел
- обчисленнями (сума, min, max)
- умовами (парні/непарні)
- сортуванням (метод обміну та метод екстремальних елементів)
- візуалізацією в DataGridView

---

## 🧩 Завдання

1. Заповнити одновимірний масив і вивести в DataGridView  
2. Знайти суму, мінімум, максимум  
3. Порахувати парні/непарні числа  
4. Заповнити матрицю 5×5  
5. Знайти суму рядків та діагональ  
6. Транспонувати матрицю  
7. Відсортувати масив методом обміну і методом екстремальних елементів
